# Ассемблер и интерпретатор УВМ  

Реализация ассемблера и интерпретатора для учебной вычислительной машины (УВМ) в рамках курсовой работы.

## Поддерживаемые команды

— **LOAD const, Rn**  
  Загрузка константы в регистр.  
  Пример: `LOAD 26, R16` → после выполнения `R16 = 26`.

— **READ Rsrc, offset, Rdst**  
  Чтение из памяти: `Rdst ← RAM[Rsrc + offset]`.  
  Пример: `READ R0, 48, R15` → `R15 = значение из ячейки 48`.

— **WRITE Rsrc, Rdst**  
  Запись в память: `RAM[Rdst] ← Rsrc`.  
  Пример: `WRITE R9, R25` → значение из `R9` записывается в ячейку с адресом из `R25`.

— **ADD Rsrc, addr, Raddr**  
  Сложение: `RAM[addr] ← Rsrc + RAM[Raddr]`.  
  Важно: `addr` — это **число (адрес назначения)**, а `Raddr` — регистр, содержащий адрес второго операнда.

Регистры: `R0` … `R31`.  
Память: 1024 ячейки (адреса от 0 до 1023).

## Файлы проекта

— `asm.py` — ассемблер (этапы 1 и 2)

— `test_add.asm` — проверка команды ADD
